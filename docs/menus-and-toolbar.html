Menus and toolbar

Main menus

On the Touch UI, the menus File, Edit, Modify, View, and Help, along with your User Account options(2), are all located in the hamburger-dropdown menu at the top-left(1).

Toolbar

The Toolbar is the place to find the main tools on the Touch interface, like Selection and Drawing tools, Shapes, Text, Knife, Freehand, Add Image, the shortcut to the Export Dialog, the File name, and the Share button. All the other options, such as Flip, Rotate, and Group, for example, were moved to the context panels. 

The File Tabs are still located in the Toolbar as well, with a dropdown(1) to display a list of all currently opened files(2).
