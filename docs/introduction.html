Introduction

What is Gravit Designer?

Gravit Designer is a powerful and easy-to-use graphic design application. It combines professional vector graphic capabilities and basic pixel-related functionalities in an elegant and flexible user interface that works in all modern browsers.

Gravit Designer provides everything you need to complete a variety of design projects, including:

  • illustrations
  • user interface design
  • web and social media graphics
  • icons
  • web page mockups
  • logos
  • infographics
  • print projects

Key features

  • Multiplatform – compatible with Mac, Windows, Linux, Chrome OS, Web, and touch devices (iOS, Android, and Windows)
  • Access from anywhere – use online and offline
  • A wide range of supported industry-standard formats – SVG, PDF, EPS, JPEG, PNG, HEIC
  • Easy to learn, powerful, context-sensitive user interface
  • Thousands of project assets, such as illustrations, icons, emojis, fonts, example files, and templates
  • Version History for files saved in the Cloud, which allows you to roll back up to 20 of your last-saved design iterations
  • Professional vector editing capabilities
    • many predefined vector primitives with advanced editing controls – rectangle, ellipse, polygon, star
    • live nested boolean operations (non-destructive)
    • state-of-the-art pen tool
  • Support for raster images
  • Real-time (live) and non-destructive editing
    • live gradients with on-canvas editing controls
    • multiple fills, borders, and effects per object
    • opacity and blending modes
  • Precision editing
  • Extended Undo history
  • RGB, HSB & CMYK color modes as global settings
  • Powerful text editing
    • access to Web (Google) and System fonts
    • upload your own custom fonts
    • line and block text
    • wide range of Character and Paragraph properties
    • attach text to path
  • Multilevel design organization
    • Pages
    • nested Layer groups
    • Groups & Frames
    • arrange, hide and lock objects
  • Reuse elements and assets
    • Symbols
    • Styles – text, and graphics
    • Swatches
    • Master pages
  • Multiple design aids
    • Rectangle & Isometric grid
    • Rulers and guidelines
    • Smart snapping options
    • Live dimensions
  • Share your Cloud files
  • Real-time design collaboration
  • Auto-save function, with the possibility to disable it at any time
  • PWA version available to install the web app and have a desktop-like feeling
